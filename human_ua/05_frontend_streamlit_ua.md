# Фронт на Streamlit

## Основний екран

### Ліва частина
- чат / журнал взаємодії;
- введення ТЗ;
- відповіді системи;
- системні уточнення;
- короткі підсумки по кроку.

### Права частина
- task summary;
- refs;
- prompt current;
- routing info;
- список attempts;
- latest outputs;
- кнопки review.

## Основні блоки
- New Task
- Current Prompt
- Outputs
- Scene Navigator
- Export Panel
