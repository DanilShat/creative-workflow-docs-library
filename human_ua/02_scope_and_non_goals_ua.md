# Scope та non-goals

## V1 scope

### Обов’язково
- локальна програма;
- фронт на Streamlit;
- метадані в локальному PostgreSQL;
- файли на локальному диску;
- browser-first інтеграції через Playwright;
- static workflow;
- базовий video workflow через сцени;
- versioning промптів;
- review loop;
- export package.

### Бажано
- scene manifests;
- теги причин відхилення;
- вибір найкращого кандидата серед результатів;
- ручне редагування промпта прямо з UI;
- очистка проміжних артефактів за retention policy.

## Non-goals для V1

- cloud deployment;
- multi-user auth;
- enterprise permissions;
- full AE integration;
- ML-based intelligent routing;
- browser automation для всіх сервісів світу;
- повний eval framework;
- автоматичний медіабанк організації.
