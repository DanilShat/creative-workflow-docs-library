# Спостережуваність, логування та майбутні eval

## Мінімум для V1
Система повинна вміти відповісти на питання:
- що саме ми пробували;
- який промпт був у цій спробі;
- який сервіс використовували;
- що вийшло;
- чому це відхилили;
- яка версія стала фінальною.

## Логувати треба
Для кожної спроби:
- provider;
- mode;
- prompt id;
- refs used;
- timestamps;
- latency;
- result path;
- review outcome.
