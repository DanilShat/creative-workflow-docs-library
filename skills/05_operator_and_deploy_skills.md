# Operator and Deploy Skills

The system and the operator workflow should support these operational skills.

## Server-side operator skills
- install and run local LLM runtime
- run FastAPI server
- run database migrations
- inspect job queue and worker status
- inspect logs, traces, and artifact records

## Worker-side operator skills
- install and run worker service
- configure browser profiles
- perform first-time logins
- verify Photoshop/After Effects bridges are online
- verify file upload/download paths

## Validation skills
- run smoke tests
- verify end-to-end static flow
- verify job lifecycle transitions
- verify failure classes like `needs_reauth`
- verify retention and cleanup behavior
